# Project Management System (Spring Boot)

This is a Spring Boot REST API project built as part of fresher training.
The system manages Projects, Teams, Members, and basic time tracking.

## Tech Stack
- Java
- Spring Boot
- Spring Web
- Lombok
- Maven

## How to Run the Project

### Prerequisites
- Java 17 (or Java 11)
- Maven
- IntelliJ IDEA (recommended)

### Steps to Run
1. Clone the repository
2. Open the project in IntelliJ
3. Wait for Maven dependencies to download
4. Run `ProjectManagementSystemApplication.java`
5. Application will start on `http://localhost:8080`

## API Endpoints (Week 1)

- GET `/api/projects`
- POST `/api/projects`
- GET `/api/teams`
- POST `/api/teams`
- GET `/api/members`
- POST `/api/members`
